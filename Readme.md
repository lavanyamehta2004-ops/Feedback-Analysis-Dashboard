# Customer Feedback Analysis Dashboard

## Overview

Developed a Power BI dashboard to analyze customer satisfaction, sales performance, footfall trends, and outlet-level performance across airport retail outlets.

## Dataset

- 20,000+ customer feedback records
- 43 airport outlets
- Sales and transaction data
- Customer satisfaction metrics

## Business Problem

Airport operators require visibility into customer experience and outlet performance to identify operational gaps and improve commercial outcomes.

## Features

- Customer Satisfaction Analysis
- Sales Performance Tracking
- Footfall Analysis
- Average Billing Amount Monitoring
- Outlet-Level Performance Comparison
- Interactive Filtering and Drill-Down Analysis

## KPIs Tracked

- Total Sales
- Total Transactions
- Total Footfall
- Average Billing Amount
- Positive Feedback %
- Customer Satisfaction Score
- Outlet Performance
- Revenue Contribution

## Tools Used

- Power BI

- Excel
- Python
- Power Query

## Dashboard Components

- KPI Cards
- Trend Analysis
- Outlet Comparison Charts
- Feedback Distribution Analysis
- Performance Scorecards
